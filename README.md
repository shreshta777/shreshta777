## About

I'm a Computer Science student specializing in Data Science, focused on building **practical, production-shaped AI systems** rather than one-off notebooks. My recent work centers on retrieval-augmented generation (RAG) pipelines — the unglamorous but critical parts: deduplicating documents at scale, chunking text so retrieval actually returns useful context, and processing messy PDFs into clean, structured data.

I'm looking for **internship and new-grad roles** in AI/ML engineering, backend engineering, or applied data science, where I can work on systems that serve real users at scale.

## Currently Building

- **Retrieval-Augmented Generation (RAG) pipelines** — from raw documents to grounded LLM answers
- **Near-duplicate document detection** using MinHash, SimHash, and Locality-Sensitive Hashing (LSH)
- **Semantic chunking strategies** that improve retrieval precision over naive fixed-size splitting
- **Search infrastructure concepts** using Azure AI Search (vector + hybrid search)
